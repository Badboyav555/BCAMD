# C Programming Basics

## Introduction to C Language

C is a **general-purpose programming language** developed by **Dennis Ritchie** at Bell Labs in 1972. It's known as the "mother of all languages" because many modern languages are derived from it.

### Why Learn C?

- 🚀 **Fast & Efficient**: Close to hardware
- 🎯 **Foundation**: Basis for C++, Java, C#
- 💻 **System Programming**: Used in OS development
- 🔧 **Embedded Systems**: Widely used in microcontrollers
